# Snake Game using SFML

A classic 2D Snake Game developed in C++ using the SFML (Simple and Fast Multimedia Library) framework. The game features smooth movement, real-time controls, collision detection, score tracking, and dynamic snake growth mechanics.

---

## Features

- Smooth snake movement
- Real-time keyboard controls
- Food spawning system
- Snake growth after eating food
- Collision detection
- Score tracking
- Game over system
- Interactive gameplay

---

## Technologies Used

- C++
- SFML
